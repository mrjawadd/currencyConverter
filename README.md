**Description:**  
A web-based **currency converter** that fetches real-time exchange rates from an API and performs accurate conversions.  
**Key Features:**  
- Supports multiple currencies  
- Live exchange rate updates  
- User-friendly interface for quick conversions  
**Tech Stack:** Python, Flask, REST API, HTML/CSS  
**Potential Improvements:** Add historical trends and graphs for better analytics.  
**GitHub:** [Currency Converter](https://github.com/mrjawadd/currencyConverter)
